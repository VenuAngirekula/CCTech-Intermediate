# Problem description

A Multi-module C++ program that reads strings as lines from standard input and print it back to stdout in sorted based on the lengths of the strings


## Usage

```bash
make clean

make
```
###Result
```bash
./sort_by_length
qwertyu
qwerty
qqwer
qwer
qwe
qw
q
Strings after sorting:
q
qw
qwe
qwer
qqwer
qwerty
qwertyu
```
