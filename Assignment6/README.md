# Description

The code will calcuate floating number arthemtic operations while taking care of Integer overflow and underflow.
  

## Usage

```shell
# Compile main cpp file

g++ Float_CalculatoR.cpp -o Float_CalculatoR

# Execute the object file

./Float_CalculatoR

```
